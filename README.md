# markdown-img-paste.nvim

Paste images from your system clipboard directly into Markdown files. 

The image is saved in an `img/` directory (relative to the current file) and a reference is inserted.

## Requirements

**wl-clipboard** (Wayland) or **xclip** (X11)

## Commands

| Command                  | Description                                      |
|--------------------------|--------------------------------------------------|
| `:PasteMarkdownImg`      | Paste image with interactive filename prompt     |
| `:PasteMarkdownImgQuick` | Paste image with auto‑generated timestamp name   |
